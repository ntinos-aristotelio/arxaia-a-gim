## Νέο στο Build 2.12 — Καρτέλα τάξης και βαθμολόγιο παραδόσεων

Το Build 2.12 κρατά την **Αποστολή τάξης** και προσθέτει ένα πιο χρήσιμο εργαλείο για τον καθηγητή: την **Καρτέλα τάξης**.

Πλέον ο κύκλος εργασίας είναι:

1. Ο καθηγητής δημιουργεί αποστολή από τα προστατευμένα εργαλεία.
2. Ο μαθητής φορτώνει τον κωδικό στο **Αποστολή τάξης**.
3. Οι ασκήσεις λύνονται μέσα στην ίδια οθόνη, με προηγούμενη / επόμενη ερώτηση.
4. Ο μαθητής πατά **Παράδοση αποτελέσματος** και παίρνει κωδικό παράδοσης.
5. Ο καθηγητής μπορεί είτε να ελέγξει έναν κωδικό από το **Έλεγχος παραδόσεων**, είτε να περάσει πολλούς κωδικούς στην **Καρτέλα τάξης**.
6. Η Καρτέλα τάξης δείχνει μαθητές, αποστολές, μέσους όρους, λάθη, αναλυτικές παραδόσεις και επιτρέπει εξαγωγή CSV.

Η λειτουργία αυτή δεν χρειάζεται server. Τα δεδομένα αποθηκεύονται τοπικά στη συσκευή του καθηγητή.

## Εργαλεία καθηγητή

Τα παρακάτω είναι μέσα στο προστατευμένο μενού **Εργαλεία καθηγητή**:

- Προεπισκόπηση καθηγητή
- Κέντρο καθηγητή
- Γεννήτρια τεστ
- Δημιουργός αποστολών
- Έλεγχος παραδόσεων
- Καρτέλα τάξης
- Λειτουργία τάξης
- Εξαγωγή προόδου
- Εισαγωγή προόδου

Ο προεπιλεγμένος κωδικός είναι:

```text
akadimia2026
```

Για αλλαγή κωδικού, άνοιξε το αρχείο `app.js` και άλλαξε τη γραμμή:

```js
const TEACHER_CODE = 'akadimia2026';
```

Σημείωση: επειδή η εφαρμογή τρέχει σε GitHub Pages χωρίς server, ο κωδικός λειτουργεί ως πρακτικό κλείδωμα μέσα στην τάξη και όχι ως απόλυτη ασφάλεια τραπεζικού τύπου.

## Ανέβασμα σε GitHub Pages

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε τα 5 αρχεία.
3. Πήγαινε Settings → Pages.
4. Επίλεξε Deploy from branch.
5. Διάλεξε main / root.
6. Άνοιξε τον σύνδεσμο που θα σου δώσει το GitHub Pages.
