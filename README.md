# Αρχαία Α΄ Γυμνασίου — PRO Quest

Παιχνιδοποιημένη εφαρμογή για GitHub Pages. Δεν ανεβάζει τις σημειώσεις ως σκέτα PDF. Τις μετατρέπει σε μαθητική εμπειρία: χάρτης, αποστολές, λέξεις-θησαυροί, quiz, XP, νομίσματα, σήματα και πρόοδος.

## Αρχεία

- `index.html`: ελάχιστο HTML, μόνο η βάση της εφαρμογής.
- `data.js`: εδώ αλλάζεις το περιεχόμενο των ενοτήτων.
- `app.js`: λογική παιχνιδιού, XP, quiz, χάρτης, σήματα.
- `styles.css`: εμφάνιση PRO.

## Ανέβασμα στο GitHub Pages

1. Δημιουργείς repository.
2. Ανεβάζεις τα 4 αρχεία.
3. Settings → Pages → Deploy from branch → main / root.
4. Άνοιγμα του link που δίνει το GitHub.

## Πώς προσθέτω ενότητα

Άνοιξε το `data.js` και αντέγραψε ένα αντικείμενο μέσα στο `lessons`. Άλλαξε: `title`, `world`, `icon`, `badge`, `hook`, `story`, `learn`, `words`, `grammar`, `missions`, `quiz`.

## Σημαντικό

Η πρόοδος αποθηκεύεται τοπικά στη συσκευή του μαθητή με `localStorage`. Δεν χρειάζεται βάση δεδομένων.
