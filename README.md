# Η Ακαδημία των Αρχαίων - Immersive Build 1

Πρώτο λειτουργικό build για GitHub Pages.

## Τι περιέχει
- Cinematic αρχική οθόνη
- Όνομα μαθητή και αποθήκευση προόδου στο browser
- Χάρτη αποστολών με περιοχές
- XP, οβολούς, σερί, κειμήλια και badges
- Micro-lessons βασισμένα στο υλικό/σημειώσεις
- Πολλαπλά είδη ασκήσεων: επιλογής, συμπλήρωσης, αντιστοίχισης, σειράς πτώσεων
- Δομή χωρίς πολλά HTML αρχεία

## Πώς αλλάζεις περιεχόμενο
Άνοιξε μόνο το `data.js` και άλλαξε/πρόσθεσε ενότητες, μαθήματα και ασκήσεις.

## Ανέβασμα στο GitHub Pages
Ανέβασε τα αρχεία στον φάκελο του repository και ενεργοποίησε Settings → Pages → Deploy from branch.
