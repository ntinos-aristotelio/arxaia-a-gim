# Η Ακαδημία των Αρχαίων - Immersive Build 1.1

Συγκεντρωμένο update χωρίς αλλαγή φιλοσοφίας και χωρίς να φορτωθεί το HTML.

## Τι προστέθηκε σε αυτή την έκδοση

- Κρατήθηκε η αισθητική του Build 1.
- Εμπλουτίστηκαν οι υπάρχουσες 5 περιοχές.
- Κάθε περιοχή έχει πλέον 8 δοκιμασίες αντί για 3.
- Προστέθηκαν περισσότερες ασκήσεις βασισμένες στο υλικό των σημειώσεων:
  - φθόγγοι, φωνήεντα, δίφθογγοι, τόνοι, πνεύματα
  - ρήμα εἰμί, ενεστώτας, παρατατικός, απαρέμφατο, μετοχή
  - ουσιαστικά Α΄ και Β΄ κλίσης
  - ρήματα βαρύτονα Α΄ συζυγίας, χρονικός χαρακτήρας, αύξηση, αναδιπλασιασμός
  - λεξιλόγιο Ενότητας 4: φωνή
  - λεξιλόγιο Ενότητας 5: γη / γεω-
- Η σφραγίδα ενότητας απαιτεί τώρα τουλάχιστον 3 λυμένες δοκιμασίες.
- Οι απαντήσεις συμπλήρωσης δέχονται και απλούστερη γραφή χωρίς τόνους.

## Αρχεία

- `index.html`: βασικός σκελετός
- `styles.css`: εμφάνιση
- `app.js`: λογική παιχνιδιού
- `data.js`: όλο το εκπαιδευτικό περιεχόμενο

## Πώς αλλάζεις υλικό

Πειράζεις κυρίως το `data.js`.
Δεν χρειάζεται να προσθέτεις HTML για κάθε άσκηση.

## GitHub Pages

Ανέβασε τα αρχεία σε repository και ενεργοποίησε:
Settings -> Pages -> Deploy from branch.
