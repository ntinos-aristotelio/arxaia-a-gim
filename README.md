## Ύλη που καλύφθηκε στην Ενότητα 7

### Κείμενο και κατανόηση

- Ο Αλέξανδρος φτάνει στο Γόρδιο.
- Θέλει να δει την άμαξα του Γορδίου.
- Ο χρησμός για τον δεσμό του ζυγού της άμαξας.
- Ο δεσμός από φλοιό κρανιάς.
- Η αδυναμία να βρεθεί αρχή και τέλος του δεσμού.
- Η αποφασιστική λύση με το ξίφος.
- Οι βροντές και το σέλας ως σημεία επιδοκιμασίας.

### Λεξιλόγιο

- παρῆλθεν
- πόθος
- ἅμαξα
- δεσμός
- ἄρχω
- ἐξευρίσκω
- τό σέλας
- θύω
- ἄλυτος
- γόρδιος δεσμός στη σημερινή χρήση

### Οικογένεια ζυγός / ζεύγνυμι

- ζεύγος
- ζυγός
- διάζευξη
- σύζευξη
- διαζύγιο
- ομόζυγος
- ζυγοστάθμιση

### Παραγωγή ουσιαστικών από ρήματα

- ὀρχοῦμαι -> ὀρχήστρα
- σημαίνω -> σήμαντρον
- ἀποβαίνω -> ἀποβάθρα
- αἰσθάνομαι -> αἰσθητήριον

### Γραμματική

- συλλαβική αύξηση στα ρήματα που αρχίζουν από σύμφωνο
- χρονική αύξηση στα ρήματα που αρχίζουν από φωνήεν / δίφθογγο
- παρατατικός και αόριστος του λύω
- παιδεύω -> ἐπαίδευον / ἐπαίδευσα
- ἡσυχάζω -> ἡσύχαζον / ἡσύχασα
- ἄρχω -> ἦρχον
- ὁρίζω -> ὥριζον
- οἰκίζω -> ᾤκιζον
- αὔξω -> ηὖξον
- μεταβολές με σ: π/β/φ + σ -> ψ, κ/γ/χ + σ -> ξ, τ/δ/θ + σ -> σ
- παρατατικός του εἰμί: ἦ/ἦν, ἦσθα, ἦν, ἦμεν, ἦτε/ἦστε, ἦσαν

## Πώς το βλέπεις

Άνοιξε το `index.html`.

Για έλεγχο καθηγητή:

1. Πάτα **Είσοδος στην Ακαδημία**.
2. Από το αριστερό panel πάτα **Προεπισκόπηση καθηγητή**.
3. Άνοιξε τη νέα περιοχή: **Φρυγία - Ενότητα 7**.

## Δομή αρχείων

- `index.html` - ο βασικός σκελετός
- `styles.css` - η εμφάνιση
- `app.js` - η λογική παιχνιδιού
