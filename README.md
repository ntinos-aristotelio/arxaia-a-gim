# Η Ακαδημία των Αρχαίων — Immersive Build 2.7

Εφαρμογή για Αρχαία Α΄ Γυμνασίου σε μορφή διαδραστικής ακαδημίας.

## Τι έχει ήδη καλυφθεί

- Ενότητες 1-10
- 16 περιοχές / αποστολές
- 365 δοκιμασίες
- κείμενο, λεξιλόγιο, γραμματική, σύνταξη
- mini boss ανά ενότητα
- τελική μεγάλη επανάληψη
- διαγνωστικό τεστ
- κέντρο επανάληψης
- πλάνο 7 ημερών
- προφίλ μαθητή
- προστατευμένα εργαλεία καθηγητή

## Νέο στο Build 2.7 — Ημερήσιες Αποστολές & Αρένα Επανάληψης

Το Build 2.7 δεν προσθέτει βιαστικά νέα ύλη. Προσθέτει λόγο επιστροφής και γρήγορη επανάληψη για τον μαθητή.

Νέα στοιχεία:

- Κουμπί **Ημερήσιες αποστολές** στο μαθητικό μενού.
- Κάθε μέρα εμφανίζονται 6 στοχευμένες δοκιμασίες.
- Με 3 λυμένες ημερήσιες δοκιμασίες ο μαθητής σφραγίζει την ημέρα.
- Η σφραγίδα δίνει XP, οβολούς και ημερήσιο σερί.
- Κουμπί **Αρένα επανάληψης** για γρήγορη μονομαχία 5, 10 ή 15 ερωτήσεων.
- Η αρένα μπορεί να δουλέψει με μικτή ύλη, άλυτες δοκιμασίες ή λάθη μαθητή.
- Κρατήθηκε και ο **Μέντορας Ακαδημίας** από το Build 2.5.

## Μαθητικό μενού

Τα εργαλεία που βλέπει ο μαθητής είναι:

- Διαγνωστικό τεστ
- Κέντρο επανάληψης
- Πλάνο 7 ημερών
- Προφίλ μαθητή
- Μέντορας Ακαδημίας
- Ημερήσιες αποστολές
- Αρένα επανάληψης
- Τυχαία άλυτη δοκιμασία

## Εργαλεία καθηγητή

Τα παρακάτω είναι πλέον μέσα στο προστατευμένο μενού **Εργαλεία καθηγητή**:

- Προεπισκόπηση καθηγητή
- Κέντρο καθηγητή
- Γεννήτρια τεστ
- Λειτουργία τάξης
- Εξαγωγή προόδου
- Εισαγωγή προόδου

Ο προεπιλεγμένος κωδικός είναι:

```text
akadimia2026
```

Για αλλαγή κωδικού, άνοιξε το αρχείο `app.js` και άλλαξε τη γραμμή:

```js
const TEACHER_CODE = 'akadimia2026';
```

Σημείωση: επειδή η εφαρμογή τρέχει σε GitHub Pages χωρίς server, ο κωδικός λειτουργεί ως πρακτικό κλείδωμα μέσα στην τάξη και όχι ως απόλυτη ασφάλεια τραπεζικού τύπου.

## Χρήση

Άνοιξε το `index.html` σε browser.

Για γρήγορο έλεγχο όλων των περιοχών:

1. Γράψε όνομα μαθητή.
2. Πάτα **Είσοδος στην Ακαδημία**.
3. Πάτα **Εργαλεία καθηγητή**.
4. Βάλε τον κωδικό.
5. Πάτα **Προεπισκόπηση καθηγητή**.

Για έλεγχο του νέου Build 2.7, δοκίμασε:

- **Ημερήσιες αποστολές**
- **Σφράγισε την ημέρα** αφού λύσεις 3 αποστολές
- **Αρένα επανάληψης** με 5 ή 10 ερωτήσεις

## GitHub Pages

1. Ανέβασε τα 5 αρχεία σε repository.
2. Πήγαινε Settings → Pages.
3. Επίλεξε Deploy from branch.
4. Άνοιξε το link που θα δώσει το GitHub.

## Σημείωση αποθήκευσης

Η πρόοδος αποθηκεύεται τοπικά στον browser του μαθητή με `localStorage`. Δεν χρειάζεται server ή βάση δεδομένων.
