# Η Ακαδημία των Αρχαίων - Immersive Build 1.2

Αυτή η έκδοση δεν ξηλώνει το προηγούμενο build. Κάνει ένα προσεκτικό βήμα: μετατρέπει κυρίως την περιοχή **Δελφοί - Το Μαντείο του εἰμί** σε πρότυπο ολοκληρωμένης ενότητας.

## Τι προστέθηκε στο Build 1.2

- Εμπλουτισμός της ενότητας για το ρήμα **εἰμί**.
- 16 δοκιμασίες στους Δελφούς αντί για 8.
- Νέος τύπος άσκησης: συμπλήρωση πίνακα.
- Νέος τύπος άσκησης: mini boss duel.
- Καλύτερη ροή ενότητας με φάσεις.
- Κουμπί **Προεπισκόπηση καθηγητή** για να βλέπεις όλες τις περιοχές χωρίς να χρειάζεται να τις ξεκλειδώσεις ως μαθητής.
- Το περιεχόμενο συνεχίζει να βρίσκεται στο `data.js`.

## Ύλη που δουλεύει πιο αναλυτικά αυτή η έκδοση

- Οριστική ενεστώτα του εἰμί.
- Οριστική παρατατικού του εἰμί.
- Απαρέμφατο: εἶναι.
- Μετοχή: ὤν, οὖσα, ὄν.
- Το εἰμί ως συνδετικό ρήμα.
- Υποκείμενο και κατηγορούμενο.
- Παράδειγμα από τη σημείωση: «Ἐγώ εἰμι ἡ ἄμπελος ἡ ἀληθινή...».

## Πώς το βλέπεις

Άνοιξε το `index.html`.

Αν θες να πας κατευθείαν στους Δελφούς, πάτα πρώτα **Είσοδος στην Ακαδημία** και μετά από το αριστερό panel πάτα **Προεπισκόπηση καθηγητή**.
