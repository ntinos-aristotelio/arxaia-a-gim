# Αρχαία Α΄ Γυμνασίου - Παιχνίδι Μάθησης

Αυτή η έκδοση δεν ανεβάζει τις σημειώσεις ως PDF για απλή ανάγνωση. Το υλικό έχει μετατραπεί σε παιχνιδοποιημένες ενότητες: ιστορίες, στόχους, λέξεις-θησαυρούς, γραμματικά κόλπα, αποστολές, quiz και μικρά παιχνίδια.

## Ανέβασμα στο GitHub Pages
1. Φτιάξτε νέο repository.
2. Ανεβάστε τα αρχεία `index.html`, `styles.css`, `app.js`, `data.js`, `README.md`.
3. Settings → Pages → Deploy from branch → main → root.

## Πώς αλλάζει το περιεχόμενο
Το βασικό αρχείο είναι το `data.js`. Εκεί προσθέτετε/διορθώνετε ενότητες, λέξεις, αποστολές και ερωτήσεις χωρίς να πειράζετε το HTML.
