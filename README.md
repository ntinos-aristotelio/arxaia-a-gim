# Ακαδημία Ηρώων των Αρχαίων Ελληνικών (Α' Γυμνασίου)

Παιχνιδοποιημένη εφαρμογή εκμάθησης Αρχαίων Ελληνικών βασισμένη στο React και Vite.

## Οδηγίες Εγκατάστασης & Εκτέλεσης τοπικά:
1. Κατεβάστε και αποσυμπιέστε το αρχείο.
2. Τρέξτε `npm install` για να εγκατασταθούν οι βιβλιοθήκες.
3. Τρέξτε `npm run dev` για να ξεκινήσει ο τοπικός server.
