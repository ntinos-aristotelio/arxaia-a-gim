## Νέο στο Build 2.10 — Βελτίωση Αποστολής Τάξης

Το Build 2.10 διορθώνει τη ροή της **Αποστολής τάξης**. Οι μαθητές δεν φεύγουν πλέον από την εργασία για να ανοίγουν κάθε άσκηση σε άλλη ενότητα. Η αποστολή λειτουργεί σαν ενιαίο quiz μέσα στην ίδια οθόνη.

Νέα συμπεριφορά:

- η ερώτηση εμφανίζεται έτοιμη για λύσιμο μέσα στην Αποστολή τάξης
- υπάρχουν κουμπιά **Προηγούμενη**, **Επόμενη** και **Επόμενη άλυτη**
- υπάρχει αριθμημένη μπάρα ερωτήσεων
- φαίνεται η πρόοδος της αποστολής
- ο μαθητής σφραγίζει την αποστολή όταν λύσει όλες τις ερωτήσεις

## Εργαλεία καθηγητή

Τα παρακάτω είναι μέσα στο προστατευμένο μενού **Εργαλεία καθηγητή**:

- Προεπισκόπηση καθηγητή
- Κέντρο καθηγητή
- Γεννήτρια τεστ
- Λειτουργία τάξης
- Εξαγωγή προόδου
- Εισαγωγή προόδου

Ο προεπιλεγμένος κωδικός είναι:

```text
akadimia2026
```

Για αλλαγή κωδικού, άνοιξε το αρχείο `app.js` και άλλαξε τη γραμμή:

```js
const TEACHER_CODE = 'akadimia2026';
```

Σημείωση: επειδή η εφαρμογή τρέχει σε GitHub Pages χωρίς server, ο κωδικός λειτουργεί ως πρακτικό κλείδωμα μέσα στην τάξη και όχι ως απόλυτη ασφάλεια τραπεζικού τύπου.

## Χρήση

Άνοιξε το `index.html` σε browser.

Για γρήγορο έλεγχο:

1. Γράψε όνομα μαθητή.
2. Πάτα **Είσοδος στην Ακαδημία**.
3. Δοκίμασε το νέο κουμπί **Διαδρομές μαθητή**.
4. Άνοιξε μια διαδρομή και πάτα **Άνοιγμα επόμενου βήματος**.

Για έλεγχο καθηγητή:

1. Πάτα **Εργαλεία καθηγητή**.
2. Βάλε τον κωδικό.
3. Πάτα **Προεπισκόπηση καθηγητή**.

## GitHub Pages

1. Ανέβασε τα 5 αρχεία σε repository.
2. Πήγαινε Settings → Pages.
3. Επίλεξε Deploy from branch.
4. Άνοιξε το link που θα δώσει το GitHub.

## Σημείωση αποθήκευσης

Η πρόοδος αποθηκεύεται τοπικά στον browser του μαθητή με `localStorage`. Δεν χρειάζεται server ή βάση δεδομένων.


## Build 2.10 - Αποστολή τάξης σε ροή quiz

Βελτίωση χρήσης:

- Το μαθητικό κουμπί **Αποστολή τάξης** φορτώνει τον κωδικό αποστολής και ανοίγει αμέσως την εργασία.
- Οι ασκήσεις λύνονται μέσα στην ίδια οθόνη.
- Ο μαθητής δεν χρειάζεται να πατά «Λύση» και να χάνεται μέσα στις ενότητες.
- Υπάρχει μπάρα με αριθμούς ερωτήσεων, ένδειξη λυμένης/άλυτης και κουμπί επόμενης άλυτης.
- Το εργαλείο καθηγητή **Δημιουργός αποστολών** παραμένει μέσα στα προστατευμένα Εργαλεία καθηγητή.

Κωδικός καθηγητή: `akadimia2026`
