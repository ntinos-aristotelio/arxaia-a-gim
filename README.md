# Η Ακαδημία των Αρχαίων — Immersive Build 2.2

Εφαρμογή για Αρχαία Α΄ Γυμνασίου σε μορφή διαδραστικής ακαδημίας.

## Τι έχει ήδη καλυφθεί

- Ενότητες 1-10
- 16 περιοχές / αποστολές
- 365 δοκιμασίες
- κείμενο, λεξιλόγιο, γραμματική, σύνταξη
- mini boss ανά ενότητα
- τελική μεγάλη επανάληψη

## Νέο στο Build 2.2

Το Build 2.2 δεν προσθέτει βιαστικά νέα ύλη. Κάνει την υπάρχουσα εφαρμογή πιο χρήσιμη για μάθημα, επανάληψη και αξιολόγηση.

### Διαγνωστικό τεστ

- Δημιουργεί 20 ερωτήσεις από την υπάρχουσα ύλη.
- Δίνει ποσοστό επιτυχίας.
- Εντοπίζει άξονες που θέλουν επανάληψη.
- Προτείνει περιοχές για ξαναδουλειά.
- Καταγράφει τα λάθη στο βιβλίο λαθών.

### Κέντρο επανάληψης

- Συγκεντρώνει τις ασκήσεις όπου ο μαθητής έκανε λάθος.
- Δείχνει άλυτες δοκιμασίες ανά άξονα ύλης.
- Επιτρέπει γρήγορη μετάβαση στη σωστή άσκηση.
- Κρατά ιστορικό διαγνωστικών.

### Πλάνο 7 ημερών

- Φτιάχνει μικρό εβδομαδιαίο πλάνο μελέτης.
- Προτείνει τι να κάνει ο μαθητής κάθε μέρα.
- Μπορεί να αντιγραφεί ή να εκτυπωθεί.

### Βιβλίο λαθών

Κάθε λάθος που γίνεται σε άσκηση ή διαγνωστικό αποθηκεύεται στον browser του μαθητή και εμφανίζεται στο Κέντρο Επανάληψης.

## Αρχεία

```text
index.html
styles.css
app.js
data.js
README.md
```

Το περιεχόμενο παραμένει στο `data.js`, ώστε να μη χαθείς στο HTML.

## Χρήση

Άνοιξε το `index.html` σε browser.

Για γρήγορο έλεγχο όλων των περιοχών:

1. Γράψε όνομα μαθητή.
2. Πάτα **Είσοδος στην Ακαδημία**.
3. Πάτα **Προεπισκόπηση καθηγητή**.
4. Δοκίμασε τα νέα κουμπιά:
   - Διαγνωστικό τεστ
   - Κέντρο επανάληψης
   - Πλάνο 7 ημερών

## GitHub Pages

1. Ανέβασε τα 5 αρχεία σε repository.
2. Πήγαινε Settings → Pages.
3. Επίλεξε Deploy from branch.
4. Άνοιξε το link που θα δώσει το GitHub.

## Σημείωση

Η πρόοδος αποθηκεύεται τοπικά στον browser του μαθητή με `localStorage`. Δεν χρειάζεται server ή βάση δεδομένων.


## Build 2.2 - Κέντρο Καθηγητή και Τάξης

Νέα στοιχεία:
- Κέντρο καθηγητή με πίνακα ύλης, άξονες και τύπους ασκήσεων ανά περιοχή.
- Γεννήτρια τεστ / φύλλου εργασίας από τις υπάρχουσες δοκιμασίες.
- Εκτύπωση τεστ με ή χωρίς κλείδα απαντήσεων.
- Λειτουργία τάξης για διαδραστικό πίνακα, με ομάδες και πόντους.
- Εισαγωγή προόδου από αρχείο JSON.
- Η εξαγωγή προόδου περιλαμβάνει πλέον και τις λυμένες ασκήσεις.

Η δομή παραμένει ίδια: `index.html`, `styles.css`, `app.js`, `data.js`, `README.md`.
