# Η Ακαδημία των Αρχαίων — Immersive Build 2.0

## Τι είναι

Διαδραστική εφαρμογή για Αρχαία Α΄ Γυμνασίου.  
Η ύλη των Ενοτήτων 1-10 έχει μετατραπεί σε αποστολές με χάρτη, XP, κειμήλια, mini boss και τελική επανάληψη.

Το Build 2.0 δεν προσθέτει βιαστικά νέα ύλη. Είναι ποιοτική αναβάθμιση της εμπειρίας.

## Τι νέο έχει το Build 2.0

- Καθοδηγούμενη ροή ανά ενότητα:
  - Προθέρμανση
  - Εργαστήριο
  - Πρόκληση
  - Mini boss
- Αναζήτηση περιοχών και ύλης στον χάρτη.
- Φίλτρα:
  - Όλες
  - Ανοιχτές
  - Σε εξέλιξη
  - Ολοκληρωμένες
- Πιο καθαρό dashboard προόδου.
- Συνολική ένδειξη προόδου μαθητή.
- Σημειώσεις μαθητή ανά ενότητα, αποθηκευμένες στον browser.
- Τυχαία άλυτη δοκιμασία.
- Εξαγωγή προόδου σε αρχείο JSON.
- Αναφορά καθηγητή ανά ενότητα.
- Εμφάνιση σωστών απαντήσεων μόνο σε λειτουργία καθηγητή.
- Εκτύπωση φύλλου εργασίας για κάθε ενότητα.

## Καλυμμένη ύλη

Η εφαρμογή καλύπτει τον βασικό κορμό των σημειώσεων για τις σχολικές Ενότητες 1-10:

1. Το ταξίδι των λέξεων
2. Η εκπαίδευση των παιδιών στην αρχαία Αθήνα
3. Επαγγέλματα των αρχαίων Αθηναίων
4. Ένα ταξίδι επιστημονικής φαντασίας
5. Ο πλούτος της αττικής γης
6. Η ομορφιά δεν είναι το παν
7. Η λύση του γόρδιου δεσμού
8. Ένα μοιραίο λάθος
9. Ανυπέρβλητα πρότυπα
10. Ο Σωκράτης για τη φιλία

Υπάρχει επίσης μεγάλη επανάληψη στις «Πύλες της Ακαδημίας».

## Δομή αρχείων

```text
index.html   -> βασικός σκελετός
styles.css   -> εμφάνιση
app.js       -> λογική εφαρμογής
data.js      -> εκπαιδευτικό περιεχόμενο
README.md    -> οδηγίες
```

Το περιεχόμενο παραμένει στο `data.js`, ώστε να μη γεμίσει το HTML.

## Πώς το ανοίγω

Άνοιξε το αρχείο:

```text
index.html
```

σε browser.

## Για καθηγητή

Πάτησε:

```text
Προεπισκόπηση καθηγητή
```

Τότε:

- ανοίγουν όλες οι περιοχές
- μπορείς να δεις σωστές απαντήσεις
- μπορείς να εκτυπώσεις φύλλα εργασίας
- μπορείς να δεις αναφορά καθηγητή ανά ενότητα

## Για GitHub Pages

1. Δημιούργησε νέο repository στο GitHub.
2. Ανέβασε τα 5 αρχεία.
3. Πήγαινε Settings → Pages.
4. Επίλεξε Deploy from branch.
5. Διάλεξε main / root.
6. Άνοιξε το link που θα δώσει το GitHub Pages.

## Σημείωση για την πρόοδο

Η πρόοδος αποθηκεύεται στο localStorage του browser.  
Δεν υπάρχει server και δεν συλλέγονται προσωπικά δεδομένα.
