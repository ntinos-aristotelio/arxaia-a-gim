## Νέο στο Build 2.8 — Διαδρομές Μαθητή

Το Build 2.8 δεν προσθέτει νέα ύλη. Οργανώνει την υπάρχουσα ύλη σε έτοιμες πορείες, ώστε ο μαθητής να μη χάνεται μέσα στις 365 δοκιμασίες.

Νέο μαθητικό κουμπί:

- **Διαδρομές μαθητή**

Περιλαμβάνει τις διαδρομές:

- Πρώτα βήματα στην Ακαδημία
- Γραμματική βάση
- Λεξιλόγιο και οικογένειες λέξεων
- Σιδηρουργείο ρημάτων
- Μετάφραση και σύνταξη
- Πριν από διαγώνισμα

Κάθε διαδρομή δείχνει:

- στόχο
- άξονες ύλης
- ποσοστό προόδου
- επόμενο βήμα
- λίστα βασικών δοκιμασιών
- σφραγίδα διαδρομής όταν ο μαθητής φτάσει τουλάχιστον 70%

## Εργαλεία καθηγητή

Τα παρακάτω είναι μέσα στο προστατευμένο μενού **Εργαλεία καθηγητή**:

- Προεπισκόπηση καθηγητή
- Κέντρο καθηγητή
- Γεννήτρια τεστ
- Λειτουργία τάξης
- Εξαγωγή προόδου
- Εισαγωγή προόδου

Ο προεπιλεγμένος κωδικός είναι:

```text
akadimia2026
```

Για αλλαγή κωδικού, άνοιξε το αρχείο `app.js` και άλλαξε τη γραμμή:

```js
const TEACHER_CODE = 'akadimia2026';
```

Σημείωση: επειδή η εφαρμογή τρέχει σε GitHub Pages χωρίς server, ο κωδικός λειτουργεί ως πρακτικό κλείδωμα μέσα στην τάξη και όχι ως απόλυτη ασφάλεια τραπεζικού τύπου.

## Χρήση

Άνοιξε το `index.html` σε browser.

Για γρήγορο έλεγχο:

1. Γράψε όνομα μαθητή.
2. Πάτα **Είσοδος στην Ακαδημία**.
3. Δοκίμασε το νέο κουμπί **Διαδρομές μαθητή**.
4. Άνοιξε μια διαδρομή και πάτα **Άνοιγμα επόμενου βήματος**.

Για έλεγχο καθηγητή:

1. Πάτα **Εργαλεία καθηγητή**.
2. Βάλε τον κωδικό.
3. Πάτα **Προεπισκόπηση καθηγητή**.

## GitHub Pages

1. Ανέβασε τα 5 αρχεία σε repository.
2. Πήγαινε Settings → Pages.
3. Επίλεξε Deploy from branch.
4. Άνοιξε το link που θα δώσει το GitHub.

## Σημείωση αποθήκευσης

Η πρόοδος αποθηκεύεται τοπικά στον browser του μαθητή με `localStorage`. Δεν χρειάζεται server ή βάση δεδομένων..
